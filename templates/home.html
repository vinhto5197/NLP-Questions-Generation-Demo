Instructions

1. Specify the number of questions relevant to the document to be asked.
2. Paste the document in the textbox. The document can be copy-pasted from anywhere. It can also include non-sentences like references in an academic paper or items in a table, but in essence it needs to be a coherent and cohesive document.
3. Generate. Check out one of the sample documents if no document.

Note:

If the specified number of questions is too high relative to the length of the document, fewer questions may be generated. As of the current development, it is recommended that the document has at least 20 times as many coherent sentences as there are questions to be generated, i.e. if 3 questions are specified, the document needs at least 60 coherent sentences.

You may notice some questions may be grammatically correct but are not meaningful. More robust rules and generation based on neural network are being developed, along with more features and better UI/UX. You can stay up to date with development by visiting the repo for this project - Thanks :)
